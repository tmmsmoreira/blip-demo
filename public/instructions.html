Find Twitch!

Instructions



When you first open the homepage, it will send you to the twitch API a query with the string 'all', because no search was specified yet.

On the search box, write a topic that you want and press "Let's Find It!". A search will be made, without a page refresh, and you will see a loading animation during the search process.

When the search is finished, a list of channels is showed. Press one, and a new page will open.

You will see a video channel playing with a viewers counter. The counting is made each 5 seconds.
